# Sample Meeting Transcripts

This folder contains sample meeting transcripts for testing the AI Meeting Intelligence Agent.

## Available Samples

| File | Meeting Type |
|------|--------------|
| project_review.txt | Project Review |
| sprint_planning.txt | Sprint Planning |
| client_meeting.txt | Client Meeting |
| retrospective.txt | Sprint Retrospective |

## How to Test

1. Launch the application.
2. Upload any transcript from this folder.
3. Click **Analyze Meeting**.
4. Review:
   - Executive Summary
   - Meeting Type
   - Health Score
   - Decisions
   - Risks
   - Recommendations
   - Action Items
   - Analytics Dashboard
5. Download JSON, CSV, or Markdown reports.

These transcripts are included solely for demonstration and evaluation purposes.
